SAMPLERC
Sample - recoded
Text of this Question or Item 
The GSS 'SAMPLE' variable is recoded to combine categories
3-4 and 6-7, so that black oversamples of a certain sampled
area are in the same revised 'SAMPLERC' category as the areas
not oversampled.

For each revised category, the category label indicates whether
the sample design was an older 'block quota' (BQ) design
or a 'full probability' (FP) design.

Note that SDA uses this 'SAMPLERC' variable as the 'STRATUM'
variable for calculating complex standard errors.  The GSS file
distributed by NORC now has a variable named 'VSTRAT' to be
used for this purpose.  However, that variable has thousands of 
categories and is not usable by SDA for calculating standard errors. 
Therefore SDA continues to use the recoded 'SAMPLE' variable 
as the 'STRATUM' variable.

Since the strata identified by 'VSTRAT' are all substrata of the
strata identified by 'SAMPLERC', the calculated standard errors
should be very similar.

RSAMPCODE
Revised sampcode
Text of this Question or Item 
This variable 'RSAMPCODE' is a revised version of the original GSS
variable 'SAMPCODE'. The only difference is that 'RSAMPCODE' divides
the 1972 cases into 2 random groups, for calculating standard errors.
Without this procedure, the 1972 cases were considered as all coming
from the same PSU in a single stratum, and they were therefore dropped
from the analysis whenever the design variables (the stratum and cluster
variables) were used to calculate standard errors or summary statistics.
.
Note that SDA uses this 'RSAMPCODE' variable as the 'CLUSTER' or 'PSU'
variable for calculating complex standard errors.  The GSS file
distributed by NORC now has a variable named 'VPSU' to be used for
this purpose.  However, that variable is intended to be used together
with 'VSTRAT' as the 'STRATUM' variable, and 'VSTRAT' is unusable by
SDA for calculating standard errors because it has thousands of categories
(as explained in the documentation for 'SAMPLERC').
.
(Missing data values were included in computations)
